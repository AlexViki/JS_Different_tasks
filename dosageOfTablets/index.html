Поcчитать общее количество таблеток за определеный период

Таблетки принимается каждый(период задать), начало приема понедельник

В будние дни принимается по 1 таблетке, в выходные по 2 ккл

Пример:

  • 1/ Mon - 0
  • 2/ Tue - 0
  • 3/ Wed - 1
  • 4/ Thu - 0
  • 5/ Fri - 0
  • 6/ Sat - 2
  • 7/ Sun - 0
